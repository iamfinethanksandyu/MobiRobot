`MobiRobot/MobiRobotView.cpp`

## 修改位置
在 `OnDraw()` 函数的末尾，大约第185-190行之后

## 修改后的代码
将上面的代码替换为：

```cpp
//������ʷ�켣
pDC->Polyline(pts, m_track.size());
delete [] pts;
//ѡ�ؾɻ���
pDC->SelectObject(pOldPen);

//显示小车当前位姿信息
CString strPose;
strPose.Format(_T("当前位姿：\nX = %.2f\nY = %.2f\n角度 = %.2f°"),
	m_pose.x, m_pose.y, m_pose.a * 180.0f / PI);

//设置文本背景为透明
pDC->SetBkMode(TRANSPARENT);
//设置文本颜色为深蓝色
pDC->SetTextColor(RGB(0, 0, 128));

//在窗口右上角显示位姿信息
CRect rect;
GetClientRect(&rect);
CRect textRect(rect.right - 200, 10, rect.right - 10, 100);
pDC->DrawText(strPose, &textRect, DT_LEFT);

//显示操作提示
CString strHelp = _T("操作说明：\n↑ 前进  ↓ 后退\n← 左转  → 右转\nL 左圆弧  R 右圆弧\n菜单：设置运动参数");
CRect helpRect(10, 10, 200, 150);
pDC->SetTextColor(RGB(0, 100, 0));
pDC->DrawText(strHelp, &helpRect, DT_LEFT);
}
```

## 修改说明

### 添加的功能：
1. **位姿显示**（右上角）
   - X坐标（像素）
   - Y坐标（像素）
   - 方向角（度数）

2. **操作提示**（左上角）
   - 键盘控制说明
   - 功能说明

### 显示效果：
- 右上角：深蓝色文字显示当前位姿
- 左上角：绿色文字显示操作说明
- 文字背景透明，不遮挡小车

## 编译步骤

### 在Visual Studio中：
1. 打开 `MobiRobot.sln`
2. 确认修改已保存
3. 选择 `Release` 配置
4. 点击"生成" → "生成解决方案"（Ctrl+Shift+B）
5. 编译成功后，新的exe在 `Release\MobiRobot.exe`

### 编译时间：
约30秒-1分钟

## 验证修改

运行程序后应该看到：

- ✅ 左上角显示：操作说明
